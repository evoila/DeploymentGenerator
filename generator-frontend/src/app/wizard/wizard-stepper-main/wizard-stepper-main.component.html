Welcome information !

This wizard tries to help to deploy single containers or even microservice architectures on Kubernetes. The wizard uses some questions of the Twelve-Factor App methodology to determine the needs of the users application. At the end the wizard presents the user a deployable YAML file for Kubernets.

During the wizard there exist no way to go back, but you can change settings at the summary at the end. If you require more information you can always click the info boxes for more information. All fields and questions you will see are mandatory and must be filled or answered. You should choose all names uniquely!
